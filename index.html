// 02. Tentang Saya

Sejak kecil, saya terpukau oleh bagaimana teknologi, terutama web, dapat menghubungkan dunia dan mewujudkan ide-ide mustahil. Perjalanan saya sebagai developer dimulai pada 2019, didorong oleh rasa ingin tahu dan keinginan untuk berkontribusi dalam evolusi digital.

Saya percaya pada kekuatan desain yang berpusat pada pengguna dan kode yang bersih dan efisien. Saya selalu antusias mempelajari teknologi baru dan mencari cara inovatif untuk memecahkan masalah. Saat ini, saya sangat tertarik pada web application juga keamanan data serta bercita-cita menjadi software engineer dan hacker.

Di luar dunia digital, saya adalah seorang santri kelas 1 I'dad Muhaffizhin ( Setara SMA ) di Ma'had Sahabat Qur'an Ibnu Mas'ud Yogyakarta. Keseimbangan ini membantu saya tetap kreatif dan berpikiran terbuka.
